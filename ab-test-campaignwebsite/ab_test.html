  1. Mebandingkan 2 grup test dan control untuk variabel continous
ada 2 metode jika data nya bertipe continous:
t-test dan Mann-Whitney U Test: Jika anda ingin membandingkan 2 variabel means pada A/B test,  t-test tepat digunakan jika data berdistribusi normal . jika tidak, the Mann-Whitney U test (a non-parametric alternative) dapat digunakan. sebelum kita cek hubungan antara kedua grup, kita harus memenuhi asumsi sebagai syarat dari metode yang akan digunakan

  • Normality

  • independence, baik variabel spend control dan test campaign bersifat independen karena tidak saling mempengaruhi
Karena dat berdistribusi normal, independent, dan data bertipe kontinyu maka kita akan menggunakan t-test. T-test ada 2 cara yang pertama independent t-test dan paired t-test (membandingkan means dari 1 grup namun beda waktu atau kondisi). Kita menggunakan independent t-test karena ingin membandingkan means dari 2 produk berbeda.

#perfor t test
import scipy.stats as stats
spend_con=kumpul[kumpul['Campaign Name']=='Control Campaign']['Spend [USD]']
spend_test=kumpul[kumpul['Campaign Name']=='Test Campaign']['Spend [USD]']
t_stat,p_value=stats.ttest_ind(spend_con,spend_test)
print(t_stat,p_value)

-2.9699915165656674 0.004326251207113227

Interpretation
Jika p value kurang dari nilai signifikansi yang diguanakan yaitu 0.05, kita dapat menolak hipotesis null dan menyimpulkan bahwa ada perbedaan means secara signifikan dari kedua grup
Jika p value lebih dari nilai signifikansi, kita tidak dapat menolak hipotesis null maka tidak ada perbedaan yang signifikan antara kedua grup

Kesimpulanya karena p value 0.004326251207113227 atau kurang dari 0.05 maka ada perbedaan yang signifikan means variabel Spend [USD] antara control dan test campaign, kemudian t_stat hasilnya -2.9699915165656674 negatif artinya means dari control grup variabel spend lebih rendah dari means dari test campaign

  1. membadingkan kedua grup untuk variabel diskrit
Biasanya diskrit menggunakan proporsi sebagai alat untuk inference statistik, namun kali ini kita juga akan membandingkan mean (rata-rata) variabel diskrit antara 2 variabel. kita juga akan menggunakan proporsi setelah ini
  • cek asumsi
  1. normalitas, sample size >=30 selalu distribusi normal karena teorama limit pusat, artinya data memenuhi untuk asumsi ini
  2. independece, antara 2 grup ini independent tidak saling mempengaruhi
untuk Discrete Variables:
1. Chi-squared Test atau Fisher’s Exact Test: These tests are used to examine whether distributions of categorical variables differ from one another. Given the small sample size, Fisher’s Exact Test might be particularly appropriate as it provides exact p-values for small sample sizes and is suitable when you have categorical data in a 2x2 table.

Proportion Tests (z-test): If you are comparing two proportions (e.g., conversion rates), this test can be applicable. However, with small sample sizes, ensure the assumptions of the z-test are not violated.
